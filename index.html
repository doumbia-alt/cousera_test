PARTIE ONE

Un package est un logiciel autonome et réutilisable qui comprend du code et des métadonnées, telles que le numéro de version actuel, le nom et les dépendances du package, qu'un développeur regroupe dans un endroit commun pour que les autres les utilisent. Les packages simplifient l'utilisation et la distribution de solutions aux problèmes courants tels que la nécessité d'un cadre commun pour développer un projet, tester les coureurs et les linters pour améliorer la qualité du code, ou introduire des outils d'apprentissage machine standard pour alimenter votre application.
